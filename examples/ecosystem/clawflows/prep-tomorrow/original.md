# Prep for Tomorrow

A quick evening routine to review what's on deck for tomorrow. Know what to expect, prepare what you can tonight, and set yourself up for a smooth morning.

## Steps

1. **Review tomorrow's calendar** — Use your **calendar skill** to pull all events, meetings, and deadlines for tomorrow. Note the first event of the day.

2. **Highlight what matters** — Pick the 1-2 most important things happening tomorrow. A big meeting, a deadline, a doctor's appointment — whatever needs the most attention.

3. **Suggest what to prep tonight** — Based on tomorrow's schedule, suggest practical things to do now:
   - Lay out clothes for an early meeting
   - Pack a bag or lunch
   - Review notes or slides for a presentation
   - Charge devices
   - Set an alarm

4. **Set tomorrow's top 3** — Ask the user: "If you could only accomplish 3 things tomorrow, what would they be?" Write them down.

5. **Pick a wake-up time** — Based on when the first event is, suggest a wake-up time that gives enough buffer for a calm morning.

## Output Format

```
Prep for Tomorrow — [day, date]

First event: 9am team standup
Last event: 4pm project review

Top priorities:
  1. Submit budget proposal before noon
  2. Call the dentist to reschedule
  3. Go for a run at lunch

Tonight:
  - Review budget proposal one more time
  - Pack gym clothes for the lunch run
  - Charge laptop

Suggested wake-up: 7:30am
  That gives you time for coffee and a calm start
  before the 9am standup.

Goodnight — you're all set for tomorrow.
```

## Notes

- Keep this light and quick. It's an evening wind-down, not a planning session.
- If tomorrow is wide open, celebrate that. Not every day needs to be packed.
- The wake-up time suggestion is just a suggestion. Don't be pushy about it.
- End on a positive note. The whole point is to go to bed feeling prepared, not stressed.
