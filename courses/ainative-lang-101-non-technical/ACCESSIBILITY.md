# Accessibility & comfortable reading

These materials are **plain Markdown** so people can use **whatever helps them**—no special app required.

**Why this matters for AINL:** comfortable reading helps learners **retain the vocabulary** (graph, **runtime**, **adapters**, **GraphPatch**) instead of only remembering a plot.

---

### Built-in wins (most readers)

- **Zoom:** Browser or editor zoom (often **Cmd/Ctrl +**) scales text without breaking layout.  
- **Reader mode:** Safari Reader, Firefox Reader View, Edge Immersive Reader reduce clutter on exported HTML if you paste Markdown into a publishing tool.  
- **Dark mode:** Many Markdown preview apps respect system theme.

---

### Dyslexia-friendly habits

- Prefer **short lines**: narrow window or enable “readable line length” in editors.  
- Choose fonts you trust—**OpenDyslexic**, **Atkinson Hyperlegible**, or system defaults with good spacing. Markdown lets the **reader** pick fonts.  
- Take **Episode breaks**: **SUNRISE-CERAMICS-STORY.md** is chunked on purpose.

---

### Screen readers & structure

Headings (`#`, `##`) mark sections for navigation. If you convert to PDF or slides, **preserve heading levels** so outline views stay usable.

---

### Print / PDF

From many editors: **Export PDF** after increasing base font to **14–16pt** for handouts.  
**PRESENTER-OUTLINE.md** + **CHEAT-SHEET** fit a **two-page** employee packet.
