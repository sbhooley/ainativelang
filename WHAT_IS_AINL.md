# AINL — Turn AI from a smart chat into a structured worker

**Full primer:** **[`docs/WHAT_IS_AINL.md`](docs/WHAT_IS_AINL.md)** — stakeholder narrative, problem/solution fit, includes, shipped capabilities (trajectory logging, Hyperspace emitter, `vector_memory` / `tool_registry`), and links. **Update that file** when this story changes; keep this root page as a short entry point for the repository top level.

**One line:** AI Native Lang (AINL) is a graph-canonical system for deterministic, multi-step AI workflows: compile to IR, execute with explicit adapters, and **optionally** validate with **`--strict`** (default compile is permissive — see [`docs/getting_started/STRICT_AND_NON_STRICT.md`](docs/getting_started/STRICT_AND_NON_STRICT.md)).
